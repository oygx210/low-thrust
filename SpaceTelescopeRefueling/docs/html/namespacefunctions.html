Converts classical orbit elements to modified equinoctial elements
Parameters:
===========
a   -- semimajor axis
e   -- eccentricity
inc -- inclination
Om  -- right ascension of ascending node
w   -- argument of perigee
nu  -- true anomaly
Returns:
========
p  -- semilatus rectum
f  -- e*cos(w+Omega)
g  -- e*sin(w+Omega)
h  -- tan(i/2)*cos(Omega)
k  -- tan(i/2)*sin(Omega)
L  -- true longitude
